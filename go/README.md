# smooai-agent (Go)

Go bindings for the `smooai-agent` framework — **planned, not yet implemented.**

The Rust implementation in [`../rust/`](../rust) is the source of truth. When a Go consumer appears, this directory will host either:

1. A pure-Go reimplementation tracking the Rust surface, or
2. A thin CGo wrapper around the Rust crate (depending on what makes sense at the time).

Track interest / progress against [SMOODEV-1314](https://smooai.atlassian.net/browse/SMOODEV-1314) and follow-on tickets.
